Здесь представлены различные парсеры для работы с такими форматами данных, как: YAML, INI, XML
1. Два десериализатора - полностью самописный и на основе готовой библиотеки - из YAML (без поддержки JSON синтаксиса в случае самописного варианта) в объект Python c сохранением комментариев в отдельный список (для самописного варианта)
2. Три сериализатора, обрабатывающие Python объекты: в INI самописный, в INI с применением автоупаковщика словарей в блоки INI, в XML. Все три сериализатора поддерживают передачу им списков комментариев для последующей вставки их в начало текста нового формата
3. Тесты всех программ
4. Программа для удобной демонстрации функционала лабораторной
